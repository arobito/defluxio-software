Archiv der Frequenzmessdaten

Unter http://data.netzsin.us findest Du das Archiv der Frequenzmessdaten. Derzeit wird nur ein Sensor exportiert, in Zukunft sollen die Datensätze aller Sensoren exportiert werden.

Die Messdaten unterliegen der ODC Open Database Lizenz. Pro Messtag wird eine Datei mit folgendem Format generiert:

# The https://netzsin.us grid frequency measurements are 
# (c) Mathias Dalheimer, .
#
# This database is made available under the Open Database License:
#   http://opendatacommons.org/licenses/odbl/1.0/.
# Any rights in individual contents of the database are licensed under
# the Database Contents License:
#   http://opendatacommons.org/licenses/dbcl/1.0/
# Please see http://opendatacommons.org/licenses/odbl/summary/ for a
# human-readable explanation of the license.
#
# Generated on 2015-04-23 12:52:55.549436393 +0000 UTC
#timestamp	reading
1405468800	50.005638
1405468802	50.003479
1405468804	50.006439
1405468806	50.005501
1405468808	50.005428
1405468810	50.001591
1405468812	49.996689
1405468814	49.993832
1405468816	49.987782
1405468818	49.984360
1405468820	49.982029

In jeder Zeile steht zunächst ein Unix-Timestamp (UTC), die gemessene Frequenz steht dahinter. Die Daten können einfach mit wget heruntergeladen werden:

$ wget -r http://data.netzsin.us

Die bislang entstandenen Analysescripte sind auf der Netsin.us Github-Seite verfügbar. Wenn Du eine coole Analyse entwickelt hast, schicke bitte eine Mail an Mathias (md@gonium.net).
