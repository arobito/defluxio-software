Archiv der Frequenzmessdaten

Unter http://data.netzsin.us findest Du das Archiv der Frequenzmessdaten. Derzeit wird nur ein Sensor exportiert, in Zukunft sollen die Datensätze aller Sensoren exportiert werden.

Für nichtkommerzielle Anwendungen können die Daten frei verwendet werden, TODO: Genaue Lizenz

Die Daten liegen im Textformat vor und können einfach mit wget heruntergeladen werden:

$ wget -r http://data.netzsin.us

Die bislang entstandenen Analysescripte sind auf der Netsin.us Github-Seite verfügbar. Wenn Du eine coole Analyse entwickelt hast, schicke bitte eine Mail an Mathias (md@gonium.net).
